# OS 学习指南

> 项目尚未完成

本项目尝试教你从工程角度学习 `操作系统`（Operating System） 开发。包括编译工具链、开发语言、以及 `OS` 相关知识。

## 前言

### 预备知识

本项目不是一个关于如何入门编程的书。它（至少）面向已经掌握一些计算机知识，但又想要了解计算机操作系统的读者。
